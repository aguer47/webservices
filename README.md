# Contacts API Project

This project is a REST API built with Node.js, Express, and MongoDB for storing and retrieving contact information.

## Features

- Get all contacts
- Get a single contact by ID
- MongoDB database connection
- Express server setup
- Ready for deployment on Render

## Technologies Used

- Node.js
- Express
- MongoDB Atlas
- dotenv
